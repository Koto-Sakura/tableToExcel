## tableToExcel v1.2.0 — 思源笔记表格导出/复制到Excel插件

### 功能

将思源笔记中的复杂表格（合并单元格、格内换行）复制到 Excel，或导出为 .xlsx 文件。

思源自带的表格复制功能无法正确处理合并单元格和换行，粘贴到 Excel 后会格式错乱（内容错列、换行丢失）。本插件解决此问题。

### 两种使用方式

| 方式 | 操作 | 适用场景 |
|------|------|---------|
| **复制到Excel** | 右键表格块图标 → 复制到Excel → Ctrl+V 粘贴到Excel | 快速粘贴，适合无合并单元格的表格 |
| **导出为Excel文件** | 右键表格块图标 → 导出为Excel文件 → 下载 .xlsx | 完美保留合并单元格和格内换行，适合复杂表格 |

### 使用方法

1. 在思源笔记中打开任意包含表格的文档
2. 点击表格左侧的 **块图标（☰）**
3. 选择：
   - **「复制到Excel」** — 然后在 Excel 中 Ctrl+V 粘贴
   - **「导出为Excel文件」** — 浏览器自动下载 .xlsx 文件

### 技术说明

- **复制到剪贴板**：克隆表格 DOM，清理后以 HTML 格式写入剪贴板。由于 Excel 剪贴板解析器不识别 `rowspan`，合并单元格展开为平铺数据。
- **导出 .xlsx**：使用 SheetJS 生成标准 Excel 文件，合并单元格通过 `<mergeCells>` 列表实现，格内换行通过 `wrapText` + `\n` 实现。

### 安装

**方式一：从集市安装**
打开思源 → 设置 → 集市 → 下载 → 找到 tableToExcel 并安装。

**方式二：手动安装**
1. 下载本 Release 的 `package.zip`
2. 解压到 `{工作空间}/data/plugins/tableToExcel/`
3. 重启思源，或在 设置 → 集市 → 已下载 中禁用再启用插件
